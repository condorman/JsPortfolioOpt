# JsPortfolioOpt

JavaScript implementation of PyPortfolioOpt-like see:  https://pyportfolioopt.readthedocs.io/en/latest/index.html

## Install

```bash
npm install JsPortfolioOpt
```

## Development Setup

```bash
npm install
python3.13 -m venv env_3.13
source env_3.13/bin/activate
pip install -U pip
pip install -r requirements.in
```
